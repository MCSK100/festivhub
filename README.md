# 🎉 FestivLink - Event Management Hub

## Features
- Glassmorphism/Neumorphism UI with doodle illustrations
- Light/Dark modes
- Landing page, Provider dashboard, Search ready
- Full backend API with MongoDB, Cloudinary gallery uploads
- Responsive, SEO-optimized

## Local Setup
```
# Terminal 1 - Frontend
cd frontend
npm install
npm run dev
# http://localhost:3000

# Terminal 2 - Backend
cd backend
cp .env.example .env
# Edit .env with MongoDB Atlas URI, Cloudinary keys
npm install
npm start
# http://localhost:5000
```

## Deploy (Free)
1. **Frontend**: `cd frontend && npx vercel --prod`
2. **Backend**: GitHub repo → Render.com Web Service, set env vars
3. **Services**: MongoDB Atlas (M0), Cloudinary free, Clerk dev keys

## Tech Stack
- Frontend: React/Vite + TailwindCSS
- Backend: Node/Express + MongoDB
- Media: Cloudinary
- Auth: Clerk
- Hosting: Vercel + Render
